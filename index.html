Inception

"Docker, Docker Compose를 활용하여 다중 컨테이너 애플리케이션을 만드는 프로젝트"

  • Nginx, WordPress + PHP-FPM, MariaDB는 각각 하나의 Dockerfile을 가집니다.
  • 모든 Dockerfile의 베이스 이미지는 Debian Bookworm-slim 버전으로 작성되었습니다.
  • Docker Networks를 통해 컨테이너 간 통신을 할 수 있도록 하였습니다.
  • Docker Volumes를 사용하여 데이터의 영속성을 보장하였습니다.

Minishell

"제한된 기능의 Bash 구현 프로젝트"

  • readline()을 통해 받는 사용자 입력을 엄격하게 파싱하여 토큰 단위로 정재.
  • 팀원이 토큰을 신뢰할 수 있도록 토큰을 출력하는 디버그모드 제공.
  • fork(), wait()등을 활용해 자식프로세스를 생성, 좀비/고아 프로세스가 발생하지 않도록 관리.
  • pipe()를 활용해 명령어 간 입력과 출력을 연결하는 파이프라인을 구현.
